    翻页组件

    Sui.Processbar(${option})

    procesbar.js

    类

    Sui.Component

    无

    Sui.Processbar

    jquery-1.7.1.js , sui.js ,suibase.js , processbar.js

    sui.processbar.css

    参数

    ${option}所能够包含的属性。 *表示该属性为必填项

    参数名(类型) 实例 说明
    *applyTo (String) 'dom'

    渲染容器的ID

    value (Number) 0

    组件的数值，默认为0

    showText (Boolean) true

    是否显示进度数值,默认为true
    customClass (Number) 'sui_porcess_red'

    自定义组件样式，默认为''

    公共方法

    方法
    查看所有方法： Sui.Processbar.html

    例子

    Example:创建一个标准Processbar组件

    折叠|展开 
    var process1 = new Sui.Processbar({
    applyTo: 'percent1',
    value:2
})
process1.setValue(83);

    Example:自定义样式

    折叠|展开 
    var process2 = new Sui.Processbar({
    applyTo: 'percent2'  ,
    customClass:'sui_process_red',
    value:45
})

    Example:没有进度数值的组件

    折叠|展开 
    var process3 = new Sui.Processbar({
    applyTo: 'percent3',
    value:78,
    showText:false
})
