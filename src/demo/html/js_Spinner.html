    Sui.form.Spinner(${option})

    spinner.js

    类

    Sui.form.TextField

    无

    Sui.form.Spinner

    jquery-1.7.1.js , sui.js ,suibase.js , textfield.js ,spinner.js

    sui.form.spinner.css

    参数

    ${option}所能够包含的属性。 *表示该属性为必填项

    参数名(类型) 实例 说明
    *applyTo(Date) 'input'

    渲染到组件的id
    value(Number) 5

    组件的值
    minValue(Number) 10

    组件最小值，默认为null
    maxValue(Number) 0

    组件最大值，默认为null
    toFixed(Number) 0 (默认)

    小数点后位数
    addValue(Number) 1 (默认)

    数字增加（减少）步长，建议为正整数，默认为1
    readOnly(Boolean) false (默认)

    是否为只读状态

    公共方法

    方法
    查看所有方法： Sui.form.Spinner.html

    例子

    Example:创建一个标准Spinner组件

    折叠|展开 
     spinner = new Sui.form.Spinner({
    applyTo: 'input',
    minValue:-100,
    maxValue:100,
    value:50,
    toFixed:0 //小数点后0位
});

    Example:调整取值范围

    折叠|展开 
    spinner.setMinValue(0);
spinner.setMaxValue(50);

    Example:禁用、启用组件

    折叠|展开 
    spinner.setReadOnly(readonly);
readonly = !readonly;

    Example:修改组件值

    折叠|展开 
    spinner.setValue(35);
