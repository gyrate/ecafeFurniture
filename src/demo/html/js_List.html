    参数

    ${option}所能够包含的属性。 *表示该属性为必填项

    data null

    组件对应的数据源

    labelField(String) 'label' (默认)

    服务器数据中与List组件选项描述值对应的属性名
    valueField(String) 'value' (默认)

    服务器数据中与List组件选项实际值所对应的属性名
    removableIcon (Boolean) 'sui_date'(默认) 是否需要删除图标组件
    selectedClass (String) ''(默认) 组件被选中的样式
    defaultClass (String) 'sui_date'(默认) 日期样式定义
    draggable (Boolean) false (默认) 是否启用拖拽，可通过拖拽交换两个子项的位置
    containment (DOM) null (默认) 限制拖放的范围DOM
    dropHoverClass(Date) ''(默认) 当被拖动对象在该对象的上面时，添加的样式
    acceptDragClass(Date) ''(默认) 哪些对象可以接受该对象放下，此时样式
    emptyToClear(Boolean) false(默认) 是否显示清空操作项
    itemWidth(Boolean) null(默认) 选项的宽度，有时候list项流式排列时会用到
    iconClass(String) ''(默认) 选项字段前的图标样式，不指定则不生成图标

    例子

    Example:创建一个标准List组件

    折叠|展开 
     store = new Sui.data.Store([
        { label : '中国', value : 'CN' },
        { removable : true, label : '美国', value : 'US'},
        { removable : true,label : '英国', value : 'EN'},
        { label : '意大利', value : 'IT'},
        { label : '加拿大',value : 'JN'},
        { label : '日本', value : 'JP'},
        { label : '俄罗斯', value : 'OR'},
        { label : '印度', value : 'IN'},
        { label : '德国', value : 'MD'},
        { label : '泰国', value : 'TS'}
    ]);

    var select = new Sui.List({
        applyTo : 'div', //将id为'div'的DOM元素渲染成List
        width : 150, //List宽度
        store : store, //组件数据
        draggable : true, //是否支持拖拽
        iconClass:'s_ico s_ico_node',//选项图标样式
        listeners : { //操作事件监听
            selected : function(e) {
                    alert(e.value);
            }
        }
    });
