    参数

    ${option}所能够包含的属性。 *表示该属性为必填项

    　
    参数名(类型) 实例 说明
    *applyTo (String) 'tableParent'

    渲染容器的ID

    width (Number) 600,'100%'

    组件的实际宽度

    height (Number) 300

    组件的实际高度，数据量大于展示区域时出现滚动条

    showTipWhenEmpty (Boolean) false(默认)

    如果没有数据显示提示

    tipWhenEmpty (String) "没有数据" (默认)

    没有数据提示消息

    fixWidth (Boolean) true (默认) 是否根据列宽计算表格宽度，默认为true
    defaultClass (String) 'sui_table' (默认) 表格的样式
    headerCellClass (String) 'headercell' (默认) 表头单元格的样式
    bodyCellClass (String) 'bodycell' (默认 表内容单元格的样式
    *store (Sui.data.store) null(默认) 表格记录数据
    columnModels (Sui.table.columnModels) null(默认)

    列头数据，列头数据
    selectType (String) 'single' 表格当前的选择模式,单选single和多选multi
    checkboxColumnIndex (Number) 0 当表格为多选模式时，checkbox所在列的索引
    checkboxVisibleColumn (String,Function) true 控制checkbox是否可见的属性
    selectedRowClass (String) 'sui_selectedrow' (默认) 选中行的样式
    mouseOverRowClass (String) 'sui_mouseroverrow' (默认) 鼠标悬浮行的样式
    editCellEvent (String) 'click' (默认) 点击编辑单元格,还是双击编辑单元格
    readOnly (Boolean) false (默认) 是否只读. 对于一些大数据量的表格,考虑性能,可以激活该属性
    ediable (Boolean) true (默认) 单元格是否可编辑. 是否可编辑是可以动态变化的
    isMultiRowHeader (Boolean) false (默认) 是否多行列头
    customRender (Object) null (默认) 自定义渲染
    defineLastRowClass (Boolean) false (默认) 是否是多行列头
    isShowContextMenu (Boolean) false (默认) 是否显示右键菜单,默认为false，即不显示
    enableColResize (Boolean) true (默认)

    是否通过拖拽改变列宽

    enableColMove (Boolean) true (默认)

    是否支持移动表格列

    colMixWidth (Number) 36 (默认)

    拖拽控制宽度时，允许列的最小宽度

    cellHeight (Number) 24 (默认)

    指定行高，如果cellHeight值为null则表示不限制行高
    oddClass (String) true (默认)

    是否为奇数行定义其他样式

    cellHeight (Number) null (默认) 指定单元格行高
    showColumnCtrlPanel (Boolean) true (默认) 是否显示列显示隐藏控制面板
