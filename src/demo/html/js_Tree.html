    树组件

    Sui.tree.Tree(${option})

    table.js

    类

    Sui.Component

    无

    Sui.tree.Tree

    jquery-1.7.1.js ,jquery-ui.js, sui.js ,suibase.js , tree.js

    sui.base.css,sui.tree.css

    参数

    ${option}所能够包含的属性。 *表示该属性为必填项

    参数名(类型) 实例 说明
    *renderTo (String) 'tree'

    渲染该组件所在的DOM元素id

    lineVisible (Boolean) false (默认)

    是否显示树结构线

    selectPath (Boolean) false (默认)

    选择节点之后,返回的Label是节点文本,还是节点所在路径的文本

    listeners (Object) { nodeClick : function(event) { var text = event.nodeText; Sui.log(text); } }

    事件监听器，可监听nodeClick、nodeChecked、afterBuildRoot、loaded、beforeContextMenu等事件

    enableDefaultContextMenu (Boolean) false (默认)

    是否允许使用右键菜单,当checkedType不是单选或默认时，enableDefaultContextMenu为true会开启[全选]右键菜单。需要导入menu.js和sui.menu.css文件
    checkedType (String) Sui.TreeSelectType.DEFAULT (默认)

    树节点的选择模式，可以为单选、多选或不出现选择框，详细请参考 Sui.TreeSelectType

    contextMenu (Sui.menu.ContextMenu) new Sui.menu.ContextMenu

    右键菜单，默认为null
