    系统消息窗口

    Sui.dialog.Msg (${option})

    MsgDialog

    dialog.js

    类

    无

    无

    SysSui.dialog.Msg

    jquery-1.7.1.js , dialog.js

    default.css , skin1.css

    参数

    ${option}所能够包含的属性。 *表示该属性为必填项

    body(string) 你的注册操作已发送。 提示内容
    sec(float)* 2.5 窗口停留时间，以秒为单位，缺省值2
    autoClose(boolean) true 是否自动关闭，缺省为true
    closeAfterFun(function) function(){...} 关闭后执行参数,只支持传入一个函数
    参数名(类型) 实例 说明

    公共方法

    方法
    打开模态窗口 openDialog()
    关闭当前模态窗口 closeDialog()
    查看所有方法： Sui.dialog.Msg.html

    例子

    Example:创建一个系统消息窗口

    new Sui.dialog.Msg({
        body:'您的申请操作已经提交,2秒后关闭窗口。'
    });

    Example:创建一个系统消息窗口

    new Sui.dialog.Msg({
        body:'您的申请操作已经提交,2秒后关闭窗口,并跳转页面。',
        closeAfterFun:function(){
            document.location.href = 'http://www.google.com.hk';
        }
    });

    Example:创建一个数据处理提示的消息窗口

    var waitingMsg = new Sui.dialog.Msg({
    body:'数据处理中，请等待...。',
    autoClose:false
});
setTimeout(function(){
    waitingMsg.content.removeClass('msg_loading').html('数据处理完成，2秒后关闭');
    setTimeout(function(){
        waitingMsg.closeDialog()
    },2000);
},1000)
// 通过 win.closeDialog();关闭窗口
