    参数

    name(String) "输入姓名" 输入字段名称描述
    defaultValue(String)* "罗玉凤" 输入字段默认值,缺省为空字符串
    fun(function) function(msg){alert(msg);} 处理输入字段值的函数

    例子

    Example:创建一个标准弹出窗口

    var prompt = new Sui.dialog.Prompt({
    name:'输入名称',
    defaultValue:'我是默认值',
    fun:function(msg){
        document.getElementById('std_prompt_result').value = msg;
    }
});
