    BQ 让项目开发更快捷

    Base Qone

    BQ是一个web应用前端框架，适用于快速构建可扩展的web项目原型，并在确定原型的基础上进行项目前端开发。BI的优势有以下几点：

    • 1. 提供了较为全面的界面框架和交互组件，方便快速组建和改造各种产品界面；
    • 2. 提供可自定义配置的javascript模块，代码冗余量低，可直接应用于web项目中；
    • 3. 样式渲染与页面结构高度分离，方便统一修改，或自定义主题；
    • 4. 产品定时反馈，用户参与协同开发，持续更新，实时解决开发过程中的遇到的问题。
