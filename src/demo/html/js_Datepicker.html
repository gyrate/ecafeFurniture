    日期组件

    Sui.Datepicker(${option})

    datepicker.js

    类

    Sui.Component

    无

    Sui.Date

    jquery-1.7.1.js , sui.js ,suibase.js , textfield.js , numberfield.js ,spinner.js ,date.js

    参数

    ${option}所能够包含的属性。 *表示该属性为必填项

    　
    *renderTo(DOM) Sui.getBody()

    该日期组件的父容器,建议维持Sui.getBody()不变
    *trigger(String) 'date-input'

    触发元素的id
    value(String) '2013-12-15'

    日期下拉组件的初始值，默认为null
    output(String) 'date-input'

    输出元素的id
    outputReadonly(Boolean) false(默认)

    输出框是否为只读
    currentDate(Date) null

    日历上的初始日期，初始化方法可以为

    dateObj = new Date();//日期为操作系统当前日期

    dateObj = new Date(dateVal);//日期格式为yyyy/MM/dd

    dateObj = new Date(year, month, date[, hours[, minutes[, s
    dateFormat(String) 'yyyy-MM-dd' (默认)

    日期格式，可以在初始化时指定，

    允许格式有 yyyy-MM-dd hh:mm:ss ,yyyy年MM月dd日 hh时mm分ss秒 等

    日期面板会根据格式的情况进行调整，比如指定dateFormat为yyyy， 则日期面板只出现年份选择器

    maxDate(Date) new Date('2099/12/31')(默认) 最大日期
    minDate(Date) new Date('1900/01/01')(默认) 最小日期
    range( Array, Function)

    range: function(year,month,date,day) {

    return day > 1; //禁用所有周一、周二

    }

    日期可选范围，接受数组与函数如 [startDate, endDate],

    推荐用['2013/12/12','2014/01/15']这种格式(范围不超过minDate、 maxDate的默认日期), 也可以用函数过滤，如

    默认为[]
    specailDatesFunction
    ( Function )

    specialDates: function (year, month, date, day){

    return day == 2;//选定所有星期二为特殊日期

    }

    		 定义特殊的负责的日期， 本质是一个函数，可以对指定的年月日星期进行过滤，返回true则断定为特殊日期,默认为null
    specialDates (Object)

    specialDates: {

    '20130929' : { 'background-color' : '#F4A551' }，

    '20130918':{ 'class','className' }

    }

    		特殊日期, key为"年10000+月100+日"(月份从1开始),值为css属性定义或样式类class定义，默认为null
    errorHandler (Function) 请参下面考代码实例 （默认） 错误事件处理函数

    例子

    Example:创建一个标准日期组件

    折叠|展开 
    window.datepicker = new Sui.Datepicker({
    renderTo:Sui.getBody(),
    trigger:'date-input',
    //dateFormat:'yyyy-MM-dd',
    //maxDate:new Date('2013/12/20')
});

    Example:自定义日期组件格式 yyyy-MM

    折叠|展开 
     datepicker.setDateFormat("yyyy-MM");

    Example:动态设置可选范围

    折叠|展开 
    datepicker.setRange(['2013-09-05','2013-09-25']);

    Example:将所有日期设置为可选

    折叠|展开 
    datepicker.activeDisableDate();

    Example:自定义报错处理函数

    折叠|展开 
    //通过errorHandler自定义错误信息处理方式，建议errorHandler在创建datepicker组件时声明
datepicker.errorHandler = function(event) {
    var msg = {
    invalid:'输入格式错误',
    outRange:'超出日期范围'
    };
    alert(msg[event.type]);
    datepicker.setValue(Sui.DateUtil.format(event.currentDate, event.dateFormat), event.currentDate);
}
