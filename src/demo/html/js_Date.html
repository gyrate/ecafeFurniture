    参数

    ${option}所能够包含的属性。 *表示该属性为必填项

    　
    currentDate(Date) null

    日历上的初始日期，初始化方法可以为

    dateObj = new Date();//日期为操作系统当前日期

    dateObj = new Date(dateVal);//日期格式为yyyy/MM/dd

    dateObj = new Date(year, month, date[, hours[, minutes[, s
    dateFormat(String) 'yyyy-MM-dd' (默认)

    日期格式，可以在初始化时指定，

    允许格式有yyyy-MM-dd hh:mm:ss ,yyyy年MM月dd日 hh时mm分ss秒 等

    日期面板会根据格式的情况进行调整，比如指定dateFormat为yyyy， 则日期面板只出现年份选择器

    defaultClass (String) 'sui_date'(默认) 日期样式定义
    hiddenClearButton (Boolean) false (默认) 是否隐藏清空按钮
    hiddenSureButton (Boolean) false (默认) 是否隐藏确定按钮
    maxDate(Date) new Date('2099/12/31')(默认) 最大日期
    minDate(Date) new Date('1900/01/01')(默认) 最小日期
    range( Array, Function)

    range: function(year,month,date,day) {

    return day > 1; //禁用所有周一、周二

    }

    日期可选范围，接受数组与函数如 [startDate, endDate],

    推荐用['2013/12/12','2014/01/15']这种格式(范围不超过minDate、 maxDate的默认日期), 也可以用函数过滤，如

    默认为[]
    specailDatesFunction
    ( Function )

    specialDates: function (year, month, date, day){

    return day == 2;//选定所有星期二为特殊日期

    }

    		 定义特殊的负责的日期， 本质是一个函数，可以对指定的年月日星期进行过滤，返回true则断定为特殊日期,默认为null
    specialDates (Object)

    specialDates: {

    '20130929' : { 'background-color' : '#F4A551' }，

    '20130918':{ 'class','className' }

    }

    例子

    Example:创建一个标准日期组件

    折叠|展开 
    window.date = new Sui.Date({
    renderTo : 'div',
    listeners : {
        'selected' : function(event) {
            alert(event.value);//点击每个可选日期后执行
        }
    },
    range:function(year,month,date,day){
        return day !== 2 ;//选择范围为非周二的任何天
    },
    currentDate: new Date( 2013,8,7,12,45,2),//设定当前日期
    specialDates: {  //设置特殊日期
        '20130929' : {
            'background-color' : '#F4A551'
        }
    },
    dateFormat:'yyyy年MM月dd日 hh时mm分ss秒'
});

    Example:自定义日期组件格式 yyyy

    折叠|展开 
    date.setDateFormat("yyyy");

    Example:自定义日期组件格式 yyyy-MM

    折叠|展开 
     date.setDateFormat("yyyy-MM");

    Example:自定义日期组件格式 yyyy-MM-dd hh:mm:ss

    折叠|展开 
    date.setDateFormat("yyyy-MM-dd hh:mm:ss");

    Example:动态设置可选范围

    折叠|展开 
    date.setRange(['2013-09-05','2013-09-25']);

    Example:将所有日期设置为可选

    折叠|展开 
    date.activeDisableDate();

    Example:动态标识今天为特殊日期

    折叠|展开 
    //将今天标红
var today = new Date();
var dayObj = {};
dayObj[''+ (today.getFullYear()*10000 + (today.getMonth()+1)*100 + today.getDate()) ] = {
    'background-color' : 'red'
};
date.setSpecialDates(dayObj);
