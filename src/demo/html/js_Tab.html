    类

    Sui.Tab

    jquery-1.7.1.js , sui.js ,suibase.js , tab.js

    sui.tab.css

    参数

    ${option}所能够包含的属性。*表示该属性为必填项

    参数名(类型) 实例 说明
    renderTo(String) 'div'

    渲染到组件的id
    menuClass(String) 'menu_item'(默认)

    菜单项的classNmae 通过该参数找到tab菜单
    contentClass(String) 'content_item'(默认)

    tab内容的className，通过该参数找到tab内容
    trigger(String) 'click'(默认)

    触发更换tab的方式，可以是click或hover
    needOperator(Boolean) true(默认)

    是否需要tab控制面板
    currentIndex(Number) 0(默认)

    当前选项的索引
    listeners(String) changeTab:function(e) { alert(e.index); }

    事件监听函数

    Example:创建一个标准的Tab组件

    折叠|展开 
    tab = new Sui.Tab({
    applyTo:'switch' ,
    menuClass:'menu_item',
    contentClass:'content_item',
    trigger:'click',
    needOperator:true,
    currentIndex:2,
    listeners:{
        changeTab:function(e) {
            //alert(e.index); tab变更时触发
        }
    }
}) ;

    Example:添加tab项

    折叠|展开 
    tab.addTabItem({
    menuText:'newTab',//菜单项的文字内容
    //index:1,tab索引值
    content:'<div style="height: 200px;">test' + Math.random() +'</div>'//
})

    Example:通过url请求获得tab内容

    折叠|展开 
    tab.addTabItem({
    menuText:'Ajax',
    url:'../../data/treefieldData.php'
})

    Example:通过url请求获得tab内容

    折叠|展开 
    tab.addTabItem({
    menuText:'Ajax',
    url:'../../data/treefieldData.php',
    needIframe:true //将请求获得的内容用iframe包裹
})

    Example:删除最后一个tab

    折叠|展开 
    tab.removeTabItem( tab.getTabCount() - 1 );
 //如果该tab可以被删除，则删除它

    Example:切换当前tab

    折叠|展开 
    tab.setCurrentIndex( tab.getCurrentIndex() + 1 );
 //跳转到下一个tab
