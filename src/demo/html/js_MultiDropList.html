    下拉多选框组件

    Sui.form.MultiDropList(${option})

    droplist.js

    类

    Sui.form.Droplist

    无

    Sui.form.MultiDropList

    jquery-1.7.1.js , sui.js ,suibase.js , textfield.js , triggerField.js ,list.js ，droplist.js

    sui.base.css ,sui.form.list.css ,sui.form.droplist.css ,sui.form.triggerfield.css

    参数

    ${option}所能够包含的属性。 *表示该属性为必填项

    参数名(类型) 实例 说明
    items(Array) [{ label : 'ABBCD', value : '1'},{ label : 'ABBCD2', value : '2' },]

    下拉多选框的选项数据

    applyTo(String) InputID

    要替换的元素Id

    labelField (String) 'label'(默认) 服务器输出的数据中与描述值对应的属性名，如数据为[{userName:'tom',id:'12',desc:'tom 编号12'}],则可定义labelField为'userName'
    valueField (String) 'value'(默认) 服务器输出的数据中与实际值对应的属性名，如数据为[{userName:'tom',id:'12',desc:'tom 编号12'}],则可定义valueField为'id'
    name (String) 'user' 提交表单时所对应的属性名
    value (String,Array) '3' 或 ['3','4','5'] 初始化组件时，组件的值
    searchable (Boolean) false(默认) 是否可搜索，默认为false
    createHiddenElement (Boolean) true(默认) 是否创建用于提交表单的标签,默认为true
    panelWidth (Number) 360 （默认） 下拉面板宽度，默认为null，搜索模式下默认为360
    itemWidth 100 (默认) 已选区域和待选区域选项的宽度，该参数只在searchable为true的模式下生效，默认为100

    Example:创建一个标准的MultiDropList组件

    applyTo : 'input',
    name : 'input',
    value :'44',
    items : [
        { label : '2013年质量体系改进项目', value : '11'},
        { label : '智慧路灯软件四期', value : '22'},
        { label : 'IT部2013年日常工作',  value : '33'},
        { label : '测试部2013年日常工作', value : '44'},
        { label : '南沙智慧家庭服务平台', value : '55'},
        { label : '南沙区一卡通系统',value : '66'}
    ]
});
